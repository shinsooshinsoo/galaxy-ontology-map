# 🌌 나만의 소중한 글들을 위한 '지식 은하계' 지도

이 프로그램은 여러분이 작성하신 마크다운(`.md`) 문서들의 태그를 분석하여, 문서들이 서로 어떻게 연결되어 있는지 한눈에 보여주는 **아름다운 우주 테마의 시각화 지도**를 만들어줍니다.

단순한 목록이 아닌, 지구와 우주선, 그리고 반짝이는 별들로 이루어진 나만의 지식 우주를 탐험해 보세요!

<br>

## ✨ 이런 기능들이 있어요!

*   **🪐 자동 별자리 생성**: 글에 적어둔 태그(`#AI`, `#인생` 등)를 읽어서 관련 글들을 하나의 별자리(UFO)로 모아줍니다. 
*   **🤖 똑똑한 자동 태그**: 태그를 깜빡하고 안 적으셨나요? 프로그램이 글 내용을 읽고 어울리는 태그를 알아서 찾아주거나 새로 만들어줍니다.
*   **✨ 신비로운 우주 유영**: 지도를 클릭하고 드래그하면 무중력 상태처럼 부드럽게 움직이는 은하계를 경험할 수 있습니다.
*   **📖 앉은 자리에서 바로 읽기**: 지도 위의 작은 별(문서)을 더블클릭하면, 우주 배경이 뿌옇게 변하며 세련된 팝업창으로 글 내용을 바로 읽을 수 있습니다.
*   **📱 핸드폰에서도 OK**: 만들어진 파일 하나만 핸드폰으로 옮기면, 이동 중에도 언제든 지도를 열고 글을 탐험할 수 있습니다.

<br>

## 📁 폴더는 이렇게 구성됩니다

*   **Documents/**: 여러분의 소중한 글들이 들어있는 폴더입니다.
*   **ontology_cluster.py**: 지도를 그려주는 마법 같은 프로그램 소스입니다.
*   **ontology_visual_map.html**: 완성된 나의 '지식 은하계' 지도 파일입니다. (이 파일을 열어보세요!)

<br>

## ✨ 지도를 만드는 방법

1.  **준비**: `Documents` 폴더 안에 지도로 만들고 싶은 메모나 마크다운(`.md`) 파일들을 넣어주세요. (글 안에 `#태그`를 포함하면 자동으로 분류됩니다.)
2.  컴퓨터에서 검은색 화면(터미널)을 열고 아래 명령어를 입력하여 준비물을 설치합니다. (처음 한 번만 하면 됩니다.)
    ```bash
    pip install -r requirements.txt
    ```
3.  지도를 새로 그리거나 업데이트하려면 아래 명령어를 입력합니다.
    ```bash
    python ontology_cluster.py
    ```
4.  폴더에 새로 생긴 **`ontology_visual_map.html`** 파일을 더블클릭해서 열어보세요!

<br>

## 🖱️ 지도 조작하는 법

### 💻 컴퓨터에서 볼 때
*   **마우스 휠**: 지도 확대 및 축소
*   **드래그**: 화면 이동 및 별자리 위치 옮기기
*   **클릭**: 해당 별과 연결된 다른 별자리들을 환하게 밝히기
*   **더블클릭**: 글 내용 읽기

### 📱 핸드폰에서 볼 때
*   **두 손가락 벌리기**: 확대 및 축소
*   **한 손가락 드래그**: 화면 이동
*   **한 번 터치**: 설명 보기 및 연결선 밝히기
*   **두 번 터치(따닥!)**: 글 내용 읽기

<br>

## 📝 글에 태그를 다는 법

프로그램이 지도를 더 잘 그릴 수 있도록, 글의 아무 곳에나 `#태그`를 적어주세요. 

- **따로 양식은 없습니다!** 글의 맨 위든, 중간이든, 맨 아래든 상관없습니다.
- `#태그`처럼 앞에 샵 기호를 붙이고 한 칸 띄우면 자동으로 인식합니다. (예: `#AI #자기계발`)
- 태그를 깜빡하셨더라도 프로그램이 알아서 내용을 분석해 지도를 만들어주니 걱정 마세요!
